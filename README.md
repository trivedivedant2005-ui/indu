# Dr. Indu Dayal Meshri Institute Website

A beautiful, modern, and fully responsive website for **Dr. Indu Dayal Meshri Institute Of Computer Science & Technology** at Hemchandracharya North Gujarat University (HNGU).

## 📋 Overview

This website is built with pure HTML5, CSS3, and Vanilla JavaScript - no frameworks required. It features a professional design with smooth animations, responsive layouts, and comprehensive functionality for a college.

## 📄 Pages Included

### 1. **Home Page** (`index.html`)
- Attractive hero section with call-to-action buttons
- Program overview for MSc CA & IT
- Why choose us section with feature cards
- Statistics display
- Quick access links
- Professional footer

### 2. **Student Facilities** (`facilities.html`)
- Comprehensive list of available facilities:
  - Computer Labs (4 labs with 60+ workstations)
  - Library & Learning Resources
  - Sports & Recreation
  - Cafeteria & Food Service
  - Transportation Services
  - Health & Medical Services
  - Wi-Fi & Connectivity
  - Academic Support
  - Hostel Accommodation
- Additional amenities section

### 3. **Student Portal** (`portal.html`)
- Student login system with validation
- Portal features overview
- Feature cards explaining portal capabilities
- 4-step guide for portal usage
- Support contact information
- FAQ section for common portal issues

### 4. **Latest Updates** (`updates.html`)
- News and announcements section with:
  - 9 sample update cards
  - Date and category labels
  - "Read More" links
- Upcoming events calendar
- Newsletter subscription form
- Professional update management

### 5. **Contact Us** (`contact.html`)
- Contact form with validation
- Multiple ways to reach the institute:
  - Phone numbers (Main, Admissions, Academic, IT Support)
  - Email addresses
  - Physical address
  - Office hours
- 6 departmental contact cards
- Google Map embed (placeholder)
- 6-item FAQ accordion with expandable answers

## 🚀 Getting Started

### Option 1: Open in Browser
Simply open any HTML file in your web browser:
- Right-click on the file → Open with Browser
- Or drag and drop the file into your browser

### Option 2: Live Server (Recommended)
If using VS Code:
1. Install the "Live Server" extension
2. Right-click on `index.html`
3. Select "Open with Live Server"

### Option 3: Host Online
Upload all files to any web hosting service:
- Netlify, Vercel, GitHub Pages, etc.
- All files should be in the same directory
- Ensure all links maintain their relative paths

## 🔧 Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1e40af;      /* Change to your primary color */
    --secondary-color: #3b82f6;    /* Change to your secondary color */
    --accent-color: #f59e0b;       /* Change to your accent color */
    /* ... other colors ... */
}
```

### Update Content
- **Institute Name**: Search and replace "Dr. Indu Dayal Meshri Institute" with your institute name
- **Contact Info**: Update phone numbers and emails in the contact page and footer
- **Course Info**: Modify course details in `index.html` and `facilities.html`
- **Address**: Update address sections in `contact.html`

### Add More Pages
1. Create a new HTML file following the same structure
2. Copy the navigation bar and footer from existing pages
3. Add the same CSS and JS references
4. Update navigation links across all pages
